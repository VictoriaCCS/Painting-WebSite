#First Web Design Project

**Showcase website for displaying and selling paintings**  

## Description
This is my **first web design project**, built to showcase and sell my artwork.  
It was created using **HTML and CSS only**, without JavaScript, and includes visual effects and styling to create an interactive user experience.  

## Features
- Responsive layout using HTML and CSS  
- Visual effects for images, buttons, and menus  
- Organized gallery to display paintings  
- Styled pages for “Home”, “Gallery”, and “About” sections  

## Tech Stack
- HTML5  
- CSS3  

## Notes
- This project is **not deployed online**; it runs locally by opening the HTML files in a web browser.  
- It demonstrates foundational web design skills, including layout, styling, and visual effects.
